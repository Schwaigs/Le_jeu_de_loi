I - Présentation du projet

Le Jeu de Lois résulte d'un projet tuteuré, ayant pour but la création d'un jeu sérieux permettant à enseigner des connaissances sur un domaine précis. Dans notre cas, l'objectif était de faire comprendre au joueur que les lois sont apparues à la suite de nombreux processus et qu'elles n'ont rien d'absolu ou d'objectif.

Vous pouvez retrouver en détail les objectifs pédagogiques dans le Wiki du projet.

IV - Réutilisation

Une archive du projet est disponible sur ce dépôt Git, elle regroupe les pages utilisées pour le projet.

Du côté de l'hébergement, nous avons utilisé celui fourni par l'IUT via webetu. Afin de reproduire ce lien entre la base PHP MyAdmin et le serveur, il faut que votre page web soit héberger, voici un hébergeur que vous pouvez choisir : https://www.one.com. Une fois votre site web hébergé, vous devrez créer une base de données sur PHP MyAdmin, voici un tutoriel vous permettant de le faire : https://openclassrooms.com/fr/courses/918836-concevez-votre-site-web-avec-php-et-mysql/913893-phpmyadmin.
